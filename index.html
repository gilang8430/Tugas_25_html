Mengenal Sematic Elements Di HTML5

  • Article

    • Ketika kita ingin membuat layout untuk menampilkan block artikel -artikel yang ada di website kita,tag article bisa digunakan dengan element header dan karena tag article memiliki block,artinya element ini memiliki lebar sesuai lebar browser jika kita tidak menentukan lebarnya.

  • aside

    • Element ini bisa kita digunakan untuk melengkapi layout utama.misal kita memiliki halaman blog maka kita bisa menambahkan aside dibagian bawah postingan Karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
